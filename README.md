🔐 Login & Register UI (HTML + CSS)


Site is live at: https://gaurav-wb.github.io/Login-Register-UI-HTML-CSS-/index.html




A clean and modern authentication UI interface built using pure HTML and CSS.
Includes both Login and Register pages with a consistent design and animated button effects.

📌 Overview

This project provides a simple front-end authentication interface featuring:

Login page
Registration page
Centered responsive layout
Gradient background design
Animated glow button effect

No backend. No JavaScript. Pure UI.




⚠️ Problems (Don’t Ignore These)

Here’s the reality—  this is not production-ready:

❌ No form validation beyond required
❌ No backend (no authentication logic at all)
❌ Password input typo (Pasword)
❌ No accessibility features (labels, ARIA)
❌ No error handling (wrong input = nothing happens)
❌ Fixed width (350px) → weak responsiveness

👉 If you present this as a “full login system,” that’s misleading. It’s just UI.

🔧 Improvements You Should Make

If you actually want this to be solid:

Add JavaScript validation
Connect to a backend (Node.js / Firebase / Django)
Replace placeholders with proper <label> elements
Add input focus states
Make layout responsive (use %, max-width)
Improve security (never store passwords in plain text)
